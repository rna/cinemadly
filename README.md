<!-- ### Setup -->

### Installation

Clone the repo with:

```sh
git clone https://github.com/rna/cinemadly
```

Instal gems with:

```sh
bundle install
```

Setup database with:

```sh
   rails db:create
   rails db:migrate
```

### Usage

Start server with:

```sh
    rails server
```

Open `http://localhost:3000/` in your browser.

### Run tests

```sh
    rpsec --format documentation
```
